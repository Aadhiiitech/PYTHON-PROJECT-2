Introduction:
This study analyzes the Titanic dataset to identify factors affecting passenger survival. It helps understand how age, gender, and class influenced survival outcomes.

Dataset Overview:
The dataset contains passenger details such as age, gender, class, fare, and survival status. It includes both numerical and categorical variables for analysis.

Methodology:
Data cleaning was performed to handle missing values and encode categorical data. Exploratory analysis and basic statistical methods were applied to study survival patterns.

Statistical Analysis:
Survival rates were compared based on gender, age, and passenger class. Descriptive statistics and correlations were used to identify key influencing factors.

Conclusion:
The results show that gender, class, and age significantly affected survival chances. Women and higher-class passengers had better survival outcomes.
